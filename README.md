# ATM-simulation

## Описание

ATM Simulator — консольное приложение на Python, моделирующее базовые операции банкомата.
Проект демонстрирует принципы работы с пользовательским вводом, условными конструкциями и управлением состоянием (баланс, PIN-код). Может использоваться как учебный пример простой финансовой системы.

## Цель проекта

* Отработка базовых конструкций Python
* Моделирование логики банковских операций
* Работа с пользовательским интерфейсом в консоли
* Формирование структурированного и читаемого кода

## Функциональность

* Просмотр текущего баланса
* Снятие средств с проверкой доступного остатка
* Пополнение счета
* Изменение PIN-кода с базовой валидацией
* Обработка некорректного ввода

## Особенности реализации

* Логика построена на цикле обработки действий пользователя
* Используются условные операторы (`if/elif/else`)
* Состояние счета хранится в переменных во время выполнения
* Интерфейс реализован через текстовое меню

## Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/AdylgazievaJarkynai/ATM-simulation
```

### Переход в директорию проекта

```bash
cd ATM_SIMULATOR-
```

### Запуск

```bash
python ATM.py
```

## Пример сценария работы

1. Пользователь запускает программу
2. Вводит PIN-код (если реализована проверка)
3. Выбирает действие из меню:

   * просмотр баланса
   * снятие средств
   * пополнение
4. Получает результат операции
5. Может продолжить работу или выйти

## Ограничения

* Данные не сохраняются между запусками программы
* Отсутствует подключение к реальным банковским системам
* Минимальная защита данных (учебный уровень)
